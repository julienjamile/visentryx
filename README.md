# Visentryx

> **Early detection of student disengagement, empowering timely intervention.**

---

## 📖 About

**Visentryx** is an AI-assisted student monitoring application designed to help educational institutions identify early signs of student disengagement. By analyzing attendance patterns, such as chronic absenteeism and repeated tardiness, the system flags students who may require additional support.

The application provides counselors and school administrators with AI-generated summaries and relevant insights to facilitate timely academic and behavioral interventions. Rather than relying solely on reactive measures, Visentryx promotes a proactive, data-driven approach to student well-being and campus support.

## 🚀 Project Brief

Visentryx is a prototype mobile application that leverages Google technologies to assist schools in monitoring student engagement. The system enables counselors to review attendance records, identify students who may need attention, and receive AI-generated summaries to support informed intervention planning.

---

## 🛠️ Google Technologies Used

- 💙 **Flutter** — Cross-platform mobile application development
- 🔥 **Firebase Authentication** — Secure user authentication
- 📂 **Cloud Firestore** — Real-time cloud database
- ☁️ **Firebase Cloud Storage** — File and media storage
- ✨ **Google Gemini API** — AI-generated student summaries and insights

---

## ✨ Key Features

- 📊 Attendance pattern monitoring
- 🚩 Early identification of at-risk students
- 🤖 AI-generated student summaries
- 👨‍🏫 Counselor monitoring dashboard
- 📈 Intervention tracking and documentation
